Задача 1:

Сумма минимумов!

С учетом 2D массива размера м и n. Ваша задача написать функцию, чтобы найти сумму минимального значения в каждой строке.
Например:
[
No1, 2, 3, 4, 5, // минимальное значение строки составляет 1
5, 6, 7, 8, 9, // минимальное значение строки составляет 5
20, 21, 34, 56, 100 евро // Минимальное значение строки составляет 20
]
Таким образом, функция должна вернуться 26, потому что сумма минимумов составляет, как 1 и 5 и 20 х 26
Примечание: Вам всегда будет предоставлен непустой массив, содержащий положительные значения.

Примеры
sumMin (1,2,3), 33,22,55 евро, 16,42,345,79,1") //1
sumMin (11,2), 323,2,55 евро) //2
sumMin (13,11,2), 3,23,23,55, 23,55,3,77, 39,7,89,94 евро) //2 3 х 3 х 7

Задача 2:

Два старейших возраста


Двух старейших возрастов функция / метод должен быть завершен. Он должен принять массив чисел в качестве аргумента и вернуть два самых высоких чисел в массиве. Возвращенное значение должно быть массивом в формате «второй старейший возраст, самый старый возраст».
Порядок пройденных номеров может быть любым порядком. Массив всегда будет включать в себя не менее 2 элементов.
Например:
twoOldestAges (No1, 2, 10, 8) // должны вернуться

Задача 3:

Самое короткое слово

Простой, учитывая строку слов, вернуть длину кратчайшего слова (ы).
Строка никогда не будет пустой, и вам не нужно учитывать различные типы данных.
Примеры
"Биткоин взять на себя мир, может быть, кто знает, возможно" -
"оказывается, случайные тестовые случаи легче, чем выписать основные из них" -
"Давайте поговорим о javascript лучший язык" --'gt; 3
"Я хочу путешествовать по миру, написав код в один прекрасный день" --'gt; 1
"Давайте все пойдут в отпуск где-нибудь очень холодно" --'gt; 2
